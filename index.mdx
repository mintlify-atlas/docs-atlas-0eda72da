    <Steps>
      <Step title="Install Prerequisites">
        Ensure you have Java 19 or later installed on your system. Patient Manager is built with Java and requires the JDK to run.
        
        ```bash
        java --version
        ```
      </Step>
      
      <Step title="Clone and Build">
        Clone the repository and build the application using Maven.
        
        ```bash
        git clone https://github.com/Br1-O/RM-patientManagerApp.git
        cd RM-patientManagerApp
        mvn clean package
        ```
      </Step>
      
      <Step title="Launch the Application">
        Run the Patient Manager application. The SQLite database will be created automatically on first launch.
        
        ```bash
        java -jar target/patientManager-1.0-SNAPSHOT.jar
        ```
        
        <Note>
          The application will create a local SQLite database in your user directory for storing patient records and clinical histories.
        </Note>
      </Step>
    </Steps>
