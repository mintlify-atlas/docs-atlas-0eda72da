## Overview

The `ServiceManager` class is the central service factory that manages and provides access to all service instances in the Patient Manager application. It initializes all services with a shared `EntityManagerFactory` for database operations.

## Class Definition

```java
package com.bo.patientmanager.service;

public class ServiceManager {
    EntityManagerFactory em = null;
    private PatientService patientService = null;
    private SessionService sessionService = null;
    private PatientRelativeRelationService patientRelativeRelationService = null;
    private RelativeObservationService relativeObservationService = null;
    private RelativeService relativeService = null;
}
```

## Constructor

### ServiceManager(EntityManagerFactory em)

Creates a new ServiceManager instance and initializes all service instances.

<ParamField path="em" type="EntityManagerFactory" required>
  The JPA EntityManagerFactory used for database operations across all services
</ParamField>

<ResponseField name="ServiceManager" type="ServiceManager">
  A new ServiceManager instance with all services initialized
</ResponseField>

**Example:**
```java
EntityManagerFactory emf = Persistence.createEntityManagerFactory("PatientManagerPU");
ServiceManager serviceManager = new ServiceManager(emf);
```

## Methods

### getEm()

Returns the EntityManagerFactory instance.

<ResponseField name="EntityManagerFactory" type="EntityManagerFactory">
  The EntityManagerFactory instance used by all services
</ResponseField>

**Example:**
```java
EntityManagerFactory emf = serviceManager.getEm();
```

---

### getPatientService()

<ResponseField name="PatientService" type="PatientService">
  The PatientService instance for CRUD operations on patients
</ResponseField>

**Example:**
```java

List<Patient> allPatients = patientService.findAll();
```

---

### getSessionService()

<ResponseField name="SessionService" type="SessionService">

<ResponseField name="RelativeObservationService" type="RelativeObservationService">

## Usage Example

Complete example showing how to initialize and use the ServiceManager:

```java
import com.bo.patientmanager.service.ServiceManager;
import com.bo.patientmanager.model.Patient;
import jakarta.persistence.EntityManagerFactory;
import jakarta.persistence.Persistence;
import java.util.List;

public class Application {
    public static void main(String[] args) {
        // Initialize EntityManagerFactory
        EntityManagerFactory emf = Persistence.createEntityManagerFactory("PatientManagerPU");
        
        // Create ServiceManager
        ServiceManager serviceManager = new ServiceManager(emf);
        
        // Access individual services
        PatientService patientService = serviceManager.getPatientService();
        SessionService sessionService = serviceManager.getSessionService();
        RelativeService relativeService = serviceManager.getRelativeService();
        
        // Use services for operations
        List<Patient> patients = patientService.findAll();
        System.out.println("Total patients: " + patients.size());
        
        // Clean up
        emf.close();
    }
}
```

## Service Architecture

The ServiceManager follows the Service Layer pattern:

- **Service Manager**: Central factory for service access
- **Service Layer**: Business logic and transaction management
- **Repository Layer**: Data access and persistence
- **Entity Manager**: JPA database operations
